Sri Venkateswara Mega Food Mart

Overview:
Sri Venkateswara Mega Food Mart is a responsive static website developed to represent a food and grocery business. The project focuses on clean UI design, structured content presentation, and smooth user interaction to showcase multiple food categories in a visually appealing and accessible manner.
This project demonstrates practical skills in front-end web development using modern, industry-relevant technologies.

Business Domains Covered:
Grocery and Food Products
Juices and Beverages
Meals and Ready-to-Eat Items
North Indian Cuisine
South Indian Cuisine
Snacks
Ice Creams

Key Features:
Fully responsive layout for desktop, tablet, and mobile devices
Well-structured navigation with category-wise sections
Smooth scroll and animation effects using AOS
Clean and consistent UI using Bootstrap components
Lightweight and fast-loading static pages

Technology Stack:
Front-End Technologies
HTML5 – Semantic markup and page structure
CSS3 – Custom styling and layout design
Bootstrap – Responsive grid system and UI components

Scripting & Libraries:
JavaScript – Basic initialization functions and navigation handling
Bootstrap AOS (Animate On Scroll) – Scroll-based animation effects

How to Run / View the Project:
Clone or download the repository.
Open the project folder.
Launch index.html in any modern web browser (Chrome, Edge, Firefox).
Note: This is a static website. No backend, database, or server configuration is required.
